# SNES Emu AI - محاكي SNES الاحترافي مع الذكاء الاصطناعي 🎮✨

<div dir="rtl">

## 📖 نظرة عامة

**SNES Emu AI** هو محاكي SNES احترافي بتصميم فاخر يجمع بين حنين الألعاب الكلاسيكية وقوة الذكاء الاصطناعي الحديث. يتميز بواجهة مستخدم داكنة كونية مستوحاة من Windows 11 مع تأثيرات glassmorphism، واكتشاف ذكي للأكواد، وإدارة شاملة لحالات الحفظ.

</div>

## ✨ المميزات الرئيسية

### 🎯 الميزات الأساسية
- **مكتبة الألعاب** - عرض شبكي وقائمة مع الفلترة والترتيب
- **نظام اللعب الذكي** - إطلاق سريع مع إعدادات مسبقة
- **أكواد مدعومة بالذكاء الاصطناعي** - اكتشاف وإدارة ذكية للأكواد
- **إدارة حالات الحفظ** - متصفح مرئي مع معاينة
- **إعدادات شاملة** - رسومات، صوت، تحكم، والمزيد
- **مساعد AI** - مساعدة تفاعلية ونصائح للألعاب

### 🎨 تصميم احترافي
- ثيم كوني داكن (indigo/purple/blue) فاخر
- واجهة مستوحاة من Windows 11 مع زوايا مستديرة
- حركات وانتقالات سلسة
- تخطيطات محسّنة لسطح المكتب
- نظام مكونات متسق وسهل الصيانة
- خلفية كونية متحركة بالنجوم ⭐

## 🛠️ التقنيات المستخدمة

- ⚛️ **React 18.3** - مكتبة واجهة المستخدم

## 🎯 الشاشات الرئيسية

### 🏠 الصفحة الرئيسية (Home)
- مركز رئيسي مع إجراءات سريعة
- الألعاب المُلعوبة مؤخراً مع صور حقيقية
- قسم "Continue Playing" مع شريط تقدم
- أزرار Quick Actions (Load/Save/Reset/Cheats)
- شريط جانبي أيمن يعرض:
  - بطاقة ترحيب مع صورة اللعبة الحالية
  - حالات الحفظ الأخيرة
  - أكواد اللعبة الحالية
  - مساعد AI في الأسفل
- خلفية كونية متحركة بالنجوم ⭐

### 📚 المكتبة
مجموعة ألعاب كاملة مع عرض شبكي/قائمة، فلترة، ترتيب، وبطاقات ألعاب مفصلة.

### 🎮 إدارة الأكواد
واجهة بتبويبين:
1. **تصفح الأكواد** - بحث، فلترة، وإدارة الأكواد الموجودة
2. **AI Finder** - اكتشاف ذكي للأكواد مع اقتراحات

### 🎲 تفاصيل اللعبة
معلومات شاملة عن اللعبة مع صورة الغلاف، البيانات الوصفية، حالات الحفظ، والأكواد النشطة.

### 🕹️ شاشة اللعب
تكوين ما قبل التشغيل مع إعدادات الفيديو المسبقة، ملفات تعريف وحدة التحكم، اختيار الأكواد، ومعاينة فتحة الحفظ.

### 💾 حالات الحفظ
مدير مرئي لحالات الحفظ مع صور المعاينة، الطوابع الزمنية، والإجراءات السريعة.

### ⚙️ الإعدادات
واجهة إعدادات متعددة الفئات تغطي:
- التفضيلات العامة
- تخصيص المظهر
- تكوين المكتبة
- ميزات الذكاء الاصطناعي
- خيارات الخصوصية

### 🤖 المساعد الذكي
واجهة محادثة تفاعلية للمساعدة في الألعاب، اكتشاف الأكواد، نصائح التحسين، وحل المشاكل.

### 🔧 الأدوات القديمة
الوصول إلى ميزات متقدمة للمستخدمين المحترفين مثل عارض الذاكرة، وحدة التحكم في التصحيح، وأدوات ROM.

## 🎨 نظام المكونات

### مكونات التخطيط
- **Sidebar** - التنقل الرئيسي مع حالات نشطة وشارات
- **TopBar** - شريط البحث وإجراءات المستخدم
- **Root** - غلاف التخطيط يجمع بين الشريط الجانبي ومنطقة المحتوى

### مكونات UI
- **Button** - متغيرات متعددة (primary, secondary, outline, ghost, danger)
- **Card** - حاوية مع تأثيرات اختيارية عند التمرير
- **Input** - إدخال نص مع دعم الأيقونات
- **Switch** - مفتاح تبديل مع تسمية ووصف
- **Badge** - مؤشرات حالة مع متغيرات
- **Modal** - نظام حوار تراكبي
- **Toast** - رسائل الإشعارات

### مكونات المحتوى
- **GameCard** - عرض اللعبة مع الغلاف والشارات والبيانات الوصفية
- **EmptyState** - عنصر نائب للقوائم الفارغة
- **LoadingState** - مؤشرات التحميل والهياكل العظمية

## 🎯 مبادئ التصميم

1. **إحساس فاخر** - مظهر احترافي ومصقول
2. **Windows 11 Native** - يتبع لغة تصميم Windows 11
3. **تباعد متسق** - محاذاة نظيفة وحشو سخي
4. **Glassmorphism** - شفافية دقيقة وضبابية خلفية
5. **انتقالات سلسة** - جميع التفاعلات متحركة
6. **Desktop-First** - محسّن لإنتاجية سطح المكتب
7. **إعادة استخدام المكونات** - مبادئ DRY في كل مكان

## 📝 ملاحظات للمطورين

- جميع الألوان تستخدم خصائص CSS مخصصة لسهولة التخصيص
- المكونات تتبع اصطلاح تسمية متسق
- الحالات (hover, active, disabled) معرّفة لجميع العناصر التفاعلية
- نظام التصميم موثق بالكامل في ملفات المكونات
- الأيقونات من Lucide React للاتساق
- ألوان الحدود تستخدم rgba مع الشفافية لتأثيرات الطبقات
- خلفية النجوم المتحركة تستخدم CSS animations
