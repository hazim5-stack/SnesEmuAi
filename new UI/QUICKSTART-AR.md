## ⚡ الطريقة الأسرع للتشغيل

### 1. تحميل المشروع

**من Figma Make:**
1. افتح المشروع "تصميم واجهة المستخدم"
2. انقر على أيقونة **القائمة** (⋮) أو زر **Download**
3. اختر **"Export as ZIP"** أو **"Download Code"**
4. احفظ الملف على جهازك

**ملاحظة:** إذا حصلت على ملف `.make` بدلاً من `.zip`، جرب تغيير الامتداد:
```bash
mv "تصميم واجهة المستخدم.make" "تصميم واجهة المستخدم.zip"
```

---

### 2. فك الضغط والتثبيت

```bash
# فك ضغط الملف
unzip "تصميم-واجهة-المستخدم.zip"

# الدخول للمجلد
cd snes-emu-ai

# تثبيت جميع الحزم
npm install
```

---

### 3. التشغيل

```bash
# تشغيل المشروع
npm run dev
```

افتح المتصفح على: **http://localhost:5173**

## 🔧 حل المشاكل السريع

### ❌ المشروع لا يعمل؟

```bash
# احذف المجلدات القديمة وأعد التثبيت
rm -rf node_modules package-lock.json
npm install
npm run dev
```

### ❌ الصور لا تظهر؟
تأكد من وجود ملف `/src/app/components/figma/ImageWithFallback.tsx`

### ❌ أخطاء في Tailwind؟
تأكد من وجود:
- `/src/styles/theme.css`
- `/src/styles/tailwind.css`
- `/src/styles/index.css`

### ❌ أخطاء في React Router؟
```bash
# تأكد من تثبيت react-router وليس react-router-dom
npm uninstall react-router-dom
npm install react-router
```
