## 🚀 ثلاث خطوات فقط!

### 1️⃣ حمّل المشروع من Figma Make

في Figma Make:
- انقر على **Download** أو **Export**
- اختر **"Export as ZIP"**

### 2️⃣ فك الضغط وافتح في Cursor

```bash
unzip "project.zip"
cd project-name
cursor .
```

### 3️⃣ ثبّت وشغّل

```bash
npm install
npm run dev
```

**⚠️ ملاحظة:** إذا كان المشروع من Figma Make، ستحتاج لإنشاء ملفين إضافيين (موضح في الأدلة التفصيلية أدناه)
